井字棋
-----
井字棋，为人们所熟知，是一种规则简单的零和博弈棋类游戏。“井字棋”因其棋盘类似“井”字而得名。井字棋通常有两种棋子：`O`和`X`。对弈双方在方寸之间着此二棋大显身手。

FIDT介绍
--------
> FIDT（Fédération Internationale des Tic-tac-toe 世界井字棋协会）是制定通用井字棋标准，维护井字棋比赛秩序，维护棋手合法权益的组织。
